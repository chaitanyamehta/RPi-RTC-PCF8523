# RTC-PCF8523
Program to get date time, day of week and battery health from PCF8523 real time clock using I2C on RPI.

## Running the program
1. Compile using make.
   ```console
   $ make
   ```
2. Run program.
   ```console
   $ ./pcf8523
   ```
